# Weather stations

The following stations are currently sending data to us
- ECOWITT HP2553
- Ecowitt HP3501
- Tempest WeatherFlow
- Davis Vantage PRO 2

These stations should also be able to send data to us (a far from complete list):
- Netatmo Weather station
- Eurochorn EFWS 2900
- WS-0101 Digital Weather Station
- Oregon" model WMR 200

**In general, any (open hardware) station to whose data you have direct access can send data to us.**
